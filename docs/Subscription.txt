    end_date TIMESTAMP,

8️⃣ Create Checkout Session

User clicks Subscribe.

Backend creates Stripe checkout.

Example:

import stripe

def create_checkout(user_id, plan_price_id):

    session = stripe.checkout.Session.create(

        payment_method_types=["card"],

        mode="subscription",

        line_items=[
            {
                "price": plan_price_id,
                "quantity": 1
            }
        ],

        success_url="https://yourapp.com/success",
        cancel_url="https://yourapp.com/cancel"

    )

    return session.url

User redirected to Stripe payment page.

9️⃣ Stripe Webhook (Important)

When payment completes Stripe sends webhook.

Flow:

Stripe
  ↓
Webhook API
  ↓
Update subscription in database

Example webhook endpoint:

@router.post("/stripe-webhook")

async def stripe_webhook(request):

    payload = await request.body()

    event = stripe.Event.construct_from(payload)

    if event["type"] == "checkout.session.completed":

        session = event["data"]["object"]

        activate_subscription(session)

    return {"status": "ok"}

Example function:

def activate_subscription(session):

    user_id = session["metadata"]["user_id"]

    subscription = Subscription(

11️⃣ Subscription Validation Middleware

Every request must verify subscription.

Example:

def check_subscription(user_id):

    sub = db.query(Subscription).filter(
        Subscription.user_id == user_id,
        Subscription.status == "active"
    ).first()

    if not sub:

        raise HTTPException(403, "Subscription required")

Used on bot APIs:

@router.post("/bot/start")

def start_bot(user=Depends(check_subscription)):
