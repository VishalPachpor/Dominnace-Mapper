1️⃣ Purpose of the Strategy Engine

The Strategy Engine is the brain of your trading bot.

It decides:
	•	when to open a trade
	•	where to place SL/TP
	•	when to move SL to break-even
	•	when to reverse trade
	•	when to stop trading

Flow:

Webhook Signal
      ↓
Strategy Engine
      ↓
Risk Manager
      ↓

5️⃣ DOM Trade Parameters

Your Pine script defines rules.

Entry

Entry = DOM candle close

Stop Loss

Bull DOM

SL = DOM low

Bear DOM

SL = DOM high

Take Profit

TP = entry ± DOM length

Example:

DOM candle
high = 50000
low = 49980
length = 20

Bull trade:

Entry = 50000
SL = 49980
TP = 50020


⸻

6️⃣ Break-Even Rule

When price reaches:

0.35 × DOM length

Move SL to entry.

Example:

entry = 50000
dom length = 20
0.35R = 7

Trigger:

price >= 50007

Then:

SL = entry


⸻

7️⃣ Reversal Rule

If SL hits before break-even:

Open opposite trade.

Example:

BUY → SL hit → open SELL

Reversal parameters:

entry = previous SL
SL = previous entry
TP = 1R


⸻

8️⃣ Standby Rule

If reversal also hits SL:

stop trading
wait for next DOM candle

Open Trade Function

def open_trade(self, symbol, action, dom_high, dom_low, close):

    dom_length = dom_high - dom_low

    if action == "buy":

        entry = close
        sl = dom_low
        tp = dom_high + dom_length
        be = entry + 0.35 * dom_length

    else:

        entry = close
        sl = dom_high
        tp = dom_low - dom_length
        be = entry - 0.35 * dom_length

    self.position = {
        "symbol": symbol,
        "side": action,
        "entry": entry,
        "sl": sl,
        "tp": tp,
        "be": be
    }

if not self.be_moved:

11️⃣ Take Profit Logic

if self.position["side"] == "buy":

    if price >= self.position["tp"]:
        close_trade()

else:

    if price <= self.position["tp"]:
        close_trade()

12️⃣ Stop Loss Logic

if self.position["side"] == "buy":

    if price <= self.position["sl"]:
        handle_stop()

else:

    if price >= self.position["sl"]:
        handle_stop()


⸻

13️⃣ Reversal Logic

def handle_stop(self):

    if not self.reversal_used:

        self.reversal_used = True
        self.reverse_trade()

    else:

        self.position = None


⸻

14️⃣ Reverse Trade

def reverse_trade(self):

    old_entry = self.position["entry"]
    old_sl = self.position["sl"]

    entry = old_sl
    sl = old_entry

Calculate TP:

risk = abs(sl - entry)

if entry > sl:
    tp = entry + risk
else:
    tp = entry - risk


⸻

15️⃣ Reset State

After TP:

def close_trade(self):

    self.position = None
    self.reversal_used = False
    self.be_moved = False
