3пёЏвѓЈ WEBHOOK SIGNAL API

Receives TradingView alerts.

Example:

from services.signal_processor import process_signal

router = APIRouter()

@router.post("/webhook")

    data = await request.json()

    process_signal(data)

    return {"status": "received"}

7пёЏвѓЈ TRADE EXECUTION ENGINE

    "apiKey": api_key,
    "secret": secret
})

    exchange.create_market_order(
        symbol=symbol,
        side=side,
        amount=amount
    )

Responsibilities:

monitor positions
